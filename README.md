# :1234:  Calculadora

Calculadora que oferece operações básicas e avalia comandos imediatamente à medida que são inseridos. 

## :arrow_right_hook: Operações

-   Adição
-   Subtração
-   Divisão
-   Multiplicação
-  Porcentagem

##  :computer:  Tecnologias usadas:

-   `React`
-   `Html`
-   `Css`
-   `Javascript`
